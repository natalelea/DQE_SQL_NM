Instructions for executing DB tests:

1. open project module4_RobotFr
2. in DB_Testing\tests\Test_MSSQL.robot file change DB parameters to your local ones
3. go to the terminal
4. execute command 'robot DB_Testing/tests/Test_MSSQL.robot' for running the tests
5. go to location module4_RobotFr\Test_Report and open png attachments to check the test report
6. optionally: go to location module4_RobotFr and open 'report.html' with any available browser to check the test report
